📊 Data Analysis Project of customer shopping behaviour
This project demonstrates an end-to-end data analysis workflow, starting from raw data processing to delivering business insights through dashboards and reports. The goal is to extract meaningful patterns, improve data quality, and present insights in a clear and actionable format.

📁 Dataset

•	The dataset contains structured records used for  customer behavior analysis .
•	It includes multiple features such as numerical, categorical, and date-related fields.
•	Data is loaded and processed using Python for further analysis.

#Tools & Technologies

•	Python (Pandas, NumPy, Matplotlib, Seaborn) – Data loading, cleaning, and EDA
•	SQL (PostgreSQL / MySQL / SQL Server) – Data querying and transformation
•	Power BI – Interactive dashboard creation
•	Jupyter Notebook – Development environment

How to Use This Project
Follow these steps to explore and use the project effectively:
1. Understand the Project Structure
•	data/ → Contains raw and cleaned datasets
•	notebooks/ → Jupyter notebooks for data analysis (EDA & cleaning)
•	sql/ → SQL scripts for querying the database
•	dashboard/ → Power BI dashboard file (.pbix)
•	reports/ → Final report and presentation

2. Run Data Analysis (Python)
•	Open Jupyter Notebook
•	Navigate to the notebooks/ folder
•	Run the notebook step by step to:
o	Load the dataset
o	Clean the data
o	Perform EDA and visualizations

3. Use SQL for Data Insights
•	Open your database tool (PostgreSQL / MySQL / SQL Server)
•	Import the cleaned dataset
•	Run queries from the sql/ folder to:
o	Analyze trends
o	Perform aggregations and joins
o	Validate insights from Python

4. Explore the Dashboard
•	Open the Power BI .pbix file
•	Interact with filters, slicers, and charts
•	Analyze key metrics and trends visually

5. Review Insights & Reports
•	Check the report in the reports/ folder for detailed analysis
•	Open the presentation (Gamma PPT) for summarized insights

6. Customize or Extend
•	Replace the dataset with your own data
•	Modify SQL queries for new business questions
•	Enhance dashboard visuals or add new KPIs
•	Extend analysis with machine learning models

7.Tips
•	Run steps in order: Python → SQL → Dashboard → Report
•	Ensure database connection settings are correct before running SQL queries
•	Use the dashboard for quick insights and the notebook for deep analysis
